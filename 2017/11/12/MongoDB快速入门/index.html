MongoDB快速入门

发表于 2017-11-12

简介


MongoDB 是一个基于分布式文件存储的数据库。由 C++ 语言编写。旨在为 WEB 应用提供可扩展的高性能数据存储解决方案。

MongoDB 是一个介于关系数据库和非关系数据库之间的产品，是非关系数据库当中功能最丰富，最像关系数据库的。

安装MongoDB

进入官网安装：https://www.mongodb.com/download-center?jmp=nav#community

配置环境变量

打开环境变量窗口进入path目录，点击新建—复制MongoDB下的bin文件夹安装路径丢进变量里面 【图示】


完成环境变量配置后，创建一个文件夹在任意盘，在桌面创建一个mongo.bat文件，里面写入mongod –dbpath d:\my_html\itheima\nodejs\文件夹路径，完成后双击桌面的bat文件即可打开mongodb数据库 【成功如图所示】
